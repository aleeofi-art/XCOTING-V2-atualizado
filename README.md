# XCoting – Ads Management SaaS

Sistema SaaS para gestão, monitoramento e contingência de contas Google Ads em escala.

## Stack
- React + Vite
- TypeScript
- Supabase
- Zustand
- Tailwind

## Rodar local

Pré-requisitos:
Node 18+

1. Instalar dependências
npm install

2. Criar .env.local
VITE_SUPABASE_URL=xxx
VITE_SUPABASE_ANON_KEY=xxx

3. Rodar
npm run dev

## Build produção
npm run build
